### YEncoder

YEncoder je jednostavan algoritam enkodiranja koji se koristi za enkodiranje binarnih podataka u ASCII format. Ovaj algoritam je ƒçesto kori≈°ƒáen u CTF takmiƒçenjima za skrivanje informacija.

Da biste dekodovali YEncoded podatke, mo≈æete koristiti razliƒçite online alate ili napisati sopstveni skript. Kada dekodujete podatke, dobiƒáete originalne binarne podatke.

Evo primera kako mo≈æete dekodovati YEncoded podatke koristeƒái Python skript:

```python
import binascii

def ydecode(data):
    decoded_data = ''
    i = 0
    while i < len(data):
        if data[i] == '=':
            break
        char = ord(data[i]) - 42
        decoded_data += chr(char)
        i += 1
    return binascii.unhexlify(decoded_data)

encoded_data = 'Y29vbCBkYXRh'
decoded_data = ydecode(encoded_data)
print(decoded_data)
```

Ovaj skript ƒáe dekodovati YEncoded podatke `'Y29vbCBkYXRh'` i ispisati originalne binarne podatke `'cool data'`.
